# ESumPretraing
員工座位表

使用方法
1. 前端
   使用command line進入esumvue資料夾
     1. npm install
     2. npm run serve
   即可啟動 Vue
2. 後端
   使用IDE進入esumspring，要使用Maven方始開啟
   透過IDE開始後即可啟動 Spring Boot
